MDX cho phép bạn dùng React component bên trong Markdown. Viết Markdown bình thường, sau đó thêm component bất cứ khi nào cần nội dung tương tác hoặc có style.

## Cú pháp Markdown

Tất cả cú pháp Markdown tiêu chuẩn đều hoạt động: heading, list, link, hình ảnh, code block, bảng, và blockquote.

### Định dạng văn bản

| Cú pháp | Kết quả |
|---------|---------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `` `code` `` | `code` |
| `[link](url)` | [link](#) |

### Code Blocks

Dùng fenced code block với language identifier để có syntax highlighting:

~~~mdx
```tsx
function Hello() {
  return <div>Hello World</div>;
}
```
~~~

## Sử dụng Components

Import và dùng React component trực tiếp trong file MDX:

```mdx
<Callout type="info" title="Note">
  Đây là callout thông tin.
</Callout>
```

<Callout type="info" title="Note">
  Đây là callout thông tin được render từ ví dụ trên.
</Callout>
