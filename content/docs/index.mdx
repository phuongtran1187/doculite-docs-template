# Welcome to Doculite

A lightweight, forkable documentation starter template.

## Features

- **MDX Content** — Write docs in MDX with full component support
- **Syntax Highlighting** — Powered by Shiki with dual theme support
- **Search** — Fast client-side search across all docs
- **Dark Mode** — Automatic theme switching with next-themes

## Quick Start

```bash
pnpm create next-app --example doculite-docs-template my-docs
```

Then start the dev server:

```bash
pnpm dev
```
