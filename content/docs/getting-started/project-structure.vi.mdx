## Thư mục chính

| Thư mục | Mục đích |
|---------|----------|
| `content/docs/` | Các file tài liệu MDX |
| `app/docs/` | Route handler Next.js cho docs |
| `components/docs/` | Component layout tài liệu |
| `components/mdx/` | Component MDX tùy chỉnh |
| `components/ui/` | shadcn/ui primitives |
| `lib/` | Hàm tiện ích và cấu hình |

## Thư mục Content

Tất cả nội dung tài liệu nằm trong `content/docs/`. Mỗi file `.mdx` trở thành một trang. Thư mục tạo nhóm navigation được điều khiển bởi file `_meta.json`.

<Callout type="info">
  Velite xử lý thư mục `content/` lúc build và tạo dữ liệu typed trong `.velite/`.
</Callout>
