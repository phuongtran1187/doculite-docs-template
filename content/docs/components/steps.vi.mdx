Steps cung cấp hướng dẫn có đánh số với đường kết nối dọc. Dùng heading `h3` bên trong component `Steps` cho mỗi bước.

## Cách dùng

<Steps>

### Tạo file mới

Thêm file MDX vào thư mục content.

### Thêm frontmatter

Thêm title và description trong khối frontmatter.

### Viết nội dung

Thêm nội dung tài liệu bằng Markdown và component.

</Steps>

~~~mdx

~~~

<Callout type="info">
  Mỗi heading `h3` bên trong Steps tự động được đánh số. Counter reset cho mỗi khối Steps.
</Callout>
